# 음성인식 리모컨

### 개발 기간
2017.09 ~ 2017.12 (4개월)

### 팀 구성 및 담당
3명 (상세설계 및 코드 구현)

### 주요 언어
- C언어
- Arduino 
	- IR센서(Infrared Ray Sensor) : 아두이노에서 IR센서용 HEX코드를 IR센서에 전송
